DISCO (C.H.U.C.K.)
==================

To load and debug the Disco, connect your PC via Wi-Fi to it and launch telnet 192.168.42.1.

copy the files to the Disco using ftp to its chrooted root (in reality /data/ftp/)


mount -o remount,rw /

create a new file /etc/boxinit.d/95-lte.rc 
# add
	service lte /data/ftp/lte.sh
	        class main

create a new file /etc/ppp/peers/lte
# change my_apn to your mobile operator's APN.
	connect "/data/ftp/chat -v -f /data/ftp/chatscript -T my_apn"
	/dev/ttyUSB1
	noipdefault
	defaultroute
	replacedefaultroute
	hide-password
	noauth
	persist
	usepeerdns

create a new file /etc/ntp.conf 
	server 0.debian.pool.ntp.org iburst
	server 1.debian.pool.ntp.org iburst
	server 2.debian.pool.ntp.org iburst
	server 3.debian.pool.ntp.org iburst

vi /etc/parameters.conf
USER_CONFIG_NO_FLY_OVER_MAX_DISTANCE                = false

mount -o remount,ro /

in glympse_gps.sh, change the following values (sorry, they are all over the place)
apikey="XXXXX" - this should be your glympse API key
/data/ftp/sendsms.sh "+XXYYYYYYYYY" [...] - this should be your phone number where you want to receive the SMS with the Glympse link - +XX is the international prefix


in lte.sh, change user, password and my_host_name to your no-ip.com account.

chmod +x /data/ftp/lte.sh
chmod +x /data/ftp/glympse_gps.sh
chmod +x /data/ftp/sendsms.sh
chmod +x /data/ftp/usb_modeswitch
chmod +x /data/ftp/pppd
chmod +x /data/ftp/chat
chmod +x /data/ftp/curl
chmod +x /data/ftp/dc

(and other binaries, if needed)
